■ツールの概要■<br>
DTM楽曲をX（旧Twitter）で魅力的に宣伝するためのカード作成ツール<br>
※スマートフォン対応可<br>

■簡単な使い方■<br>
ジャケット画像を読み込む→項目を埋める→保存してポスト！

■こだわりポイント■<br>
「テンプレート切り替え機能」「タイムスタンプ自動生成」でアピールしていきましょう

# 音画 -ONGA- Premium 🎧

DTM楽曲を X（旧Twitter）で魅力的に宣伝するための、宣伝カード作成ツールです。  
こだわりを可視化して、あなたの楽曲の魅力を1枚の画像に凝縮します。

🚀 [ツールを開く](https://ankakepizza551.github.io/music_-review/)

---

## ✨ 特徴

**3種類のデザインテンプレート**  
曲調に合わせて選べる、こだわりのデザインテンプレートを収録。

| テンプレート | イメージ | 向いている曲調 |
|------------|---------|-------------|
| 📰 標準 | マガジン風・ゴールドアクセント | ポップス・バラード・幅広いジャンル |
| 🎞 エモ系 | 映画ポスター風・モノクロフィルター | エモ・シューゲイザー・バラード |
| ⚡ 音圧/電波 | グリッチアート風・サイバー | EDM・電波系・ハードコア |

**こだわりチップ（最大3件）**  
「サビ」「重低音」「中毒性」など、アピールしたいポイントを選ぶだけで画像に自動合成。  
3件選択後は残りのチップが無効になり、迷わず選べます。

**ポスト用テキスト自動生成**  
ジャケット画像がなくても、入力内容からXポスト用テキストをリアルタイム生成。  
タイムスタンプを設定すれば「聴きどころ」も自動で反映されます。

**ワンタップ保存・シェア**  
- 「画像を保存」ボタンでダウンロード（PC・スマホ両対応）  
- スマホでは「𝕏 シェア」ボタンから画像＋テキストをそのままポスト可能（Web Share API対応端末）

---

## 🛠 使い方

1. ジャケット画像を読み込む
2. デザインテンプレートを選ぶ
3. 曲名・キャッチコピー・こだわり・タイムスタンプ・配信URLを入力
4. 「画像を保存」または「𝕏 シェア」でポスト！

---

## 🔒 プライバシーについて

本ツールはすべての処理をブラウザ内（ローカル）で完結しています。  
読み込んだ画像や入力したテキストがサーバーへ送信されることは一切ありません。

音画 -ONGA- Premium 🎧
DTM楽曲をX（旧Twitter）で魅力的に宣伝するための宣伝カード作成ツールです。
こだわりを可視化して、あなたの楽曲の魅力を1枚の画像に凝縮します。

🚀 [(https://ankakepizza551.github.io/music_-review/)]

■ 特徴
テンプレート切り替え: 曲調に合わせて「標準」「エモ系」「音圧/電波」からデザインを選択可能。

タイムスタンプ対応: 楽曲の「聴きどころ」をポスト用テキストに自動反映。

こだわりチップ: 「サビ」「重低音」など、アピールしたいポイントを視覚的に合成。

スマホ対応: ブラウザから手軽に作成・保存が可能。

■ 使い方
ジャケット画像を読み込む。

曲名、キャッチコピー、こだわりなどの項目を埋める。

生成された画像を保存し、テキストをコピーしてXへポスト！

■ プライバシーについて
本ツールはすべての処理をブラウザ内（ローカル）で行います。読み込んだ画像や入力したテキストがサーバーへ送信されることはありません。
